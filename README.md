## 🛠️ 기술 스택

**Language |** Java 11

**Framework |** Spring Boot 2.7.10

**Build Tool |** Gradle

**Database |** AWS RDS(MySQL), Redis

**ORM |** JPA(Spring Data JPA)

**Infra |** AWS EC2, AWS S3

**CI/CD |** Github Action, AWS S3, CodeDeploy

**Docs |** Swagger
